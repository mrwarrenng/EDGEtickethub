How to Use

🎯 What is this?

The Ticket Hub tracks bugs, feature requests, and content improvements for the Medford EDGE economic development platform.

📝 Submitting a Ticket

1
Click the gold 📝 Submit Ticket button in the top bar
2
Choose: Bug (broken), Feature (new), or Content (needs updating)
3
Select the EDGE page and section where you noticed it
4
Describe the issue — the system checks for duplicates so you can upvote instead

👍 Voting & Priority

See a matching ticket? Upvote it instead of creating a duplicate. More votes = higher priority. One vote per person.

📊 Dashboard

Use the view toggles (Priority / All / Recent) and type/status filters to find tickets. Click any ticket for full details and activity history.

💬 Questions?

Reach the Economic Development team at econdev@cityofmedford.org or (541) 774-2007. Click the i in the top-right of the dashboard anytime to reopen this guide.
